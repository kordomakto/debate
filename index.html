⚙️ إعداد Firebase مطلوب

تحتاج إلى إدخال إعدادات Firebase الخاصة بمشروعك في الكود.

  1. افتح console.firebase.google.com
  2. أنشئ مشروعاً → Realtime Database → Create → Test mode
  3. Project Settings ⚙️ → Your apps → </> → Register → انسخ firebaseConfig
  4. افتح index.html بـ Notepad واستبدل قيم الـ config
💡 Firebase Spark Plan مجاني تماماً — لا يحتاج بطاقة ائتمان.
